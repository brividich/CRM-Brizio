# Documentazione Tecnica

La cartella `doc/` contiene solo documentazione tecnica compatibile con un repository pubblico.

Documenti mantenuti:

- `GUIDA_AUTOMAZIONI_DESIGNER.html`
- `GUIDA_MODULI_PROGRAMMA.html`
- `STRUTTURA_ATTUALE_PORTALE.md`

Criteri applicati:

- esclusi i documenti operativi interni, i report, i file di review e le mappe di navigazione dettagliate
- esclusi i file generati o i form di raccolta requisiti non necessari al codice
- mantenuti solo documenti utili a capire struttura e funzionalita' del progetto

Note:

- i file di configurazione reali non sono inclusi nel repository
- eventuali valori presenti nei documenti devono essere interpretati come esempi o placeholder
